## Overview
AstraNotes is a high-performance, secure, and cross-platform note-taking application built in C++. Unlike standard apps, AstraNotes is designed for engineers who require markdown support, local-first encryption, and a modular architecture that can support "plugins" (Text, Voice, and Secure Notes).
